Friend Expenses V1.0


This application helps you to calculate the distribution and amout of refund in a group of users.

You can use severals commands to manage Friend Expenses :

"./friends-expenses" and "./friends-expenses --help" are used for displaying the Help.

"./friends-expenses --file FileName" is the main function, it's used to calculate and display all the data stored in your file.

"./friends-expenses --name YourName --phone YourPhone --expense YouExpense --group YourGroup --type YourType --file FileName" is used for adding a new row to your CSV file. 
It will also run the program (as ./friends-expenses --file FileName does).
